## scrapy_google_commits
- 爬取的google应用商店首页网站  
[https://play.google.com/store/apps?hl=zh_TW]

- 安装所需库  
```
pip install -U pip
pip install google-play-scraper pandas
```
- 运行  
```
python app.py
```
